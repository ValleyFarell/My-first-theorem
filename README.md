# Связность в мультипликативной модели случайного графа

В этом репозитории представлена моя теорема о связности
мультипликативной модели случайного графа и её доказательство.

## Модель
Рассматривается случайный граф на n вершинах.
Каждой вершине i приписывается независимый неотрицательный вес ω_i.

Для любых различных вершин i ≠ j ребро {i,j} проводится независимо
с вероятностью

    P({i,j} — ребро | ω_i, ω_j) = ω_i · ω_j

(или min{1, ω_i · ω_j}, если произведение может превышать 1).

## Основной результат
Теорема даёт достаточное условие связности в терминах поведения
распределения весов вблизи нуля.

Пусть F_ω(x) = P(ω ≤ x) — функция распределения веса вершины.
Если при x → 0 выполняется оценка

    F_ω(x) = O( x^2 / |ln x|^B )

для некоторого B > 1, то мультипликативный случайный граф G(n, {ω_i})
является связным
с вероятностью, стремящейся к 1 при n → ∞.

## Идея доказательства
Доказательство основано на сравнении с моделью Эрдо̋ша—Реньи G(n,p).

Для любого c > 1 с высокой вероятностью все веса вершин превосходят порог

    q = sqrt( c · ln n / n ).

На этом событии вероятность появления любого ребра не меньше

    p = c · ln n / n.

Следовательно, условная модель стохастически доминирует
случайный граф Эрдо̋ша—Реньи G(n,p).
Связность тогда следует из классической теоремы о пороге связности
для G(n,p) при c > 1.
